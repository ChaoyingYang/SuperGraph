1. The naming method of the py file is “dataset-method”, as shown in below. For example, “Gearbox-GCN.py” means that the experiments were conducted on the dataset Gearbox using GCN-based fault diagnosis method.

2. Usage
Download the datasets;
Modify the file storage address in the code;
Run the codes.
